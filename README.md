# Triburile Scavenging Distribution Helper

Un userscript pentru Tampermonkey creat pentru jucătorii de Triburile.ro / Triburile.net, care ajută la calcularea și completarea distribuției trupelor pe pagina de **Curățare**.

Scriptul detectează trupele disponibile, verifică ce niveluri de curățare sunt deblocate și distribuie trupele în ordinea:

```text
Extremă → Mare → Medie → Mică
